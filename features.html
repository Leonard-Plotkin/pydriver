getFeatures(self, dict scene, PCLHelper keypointCloud) → tuple¶

Extract features from scene at given keypoints

Returned keypoints may be identical to keypointCloud but can also differ (e.g. if the feature ignores the original y-coordinate of keypoints). The number of returned keypoints is equal or less than the number of given keypoints (some keypoints may produce invalid features).

Features is a FLOAT_dtype array of shape (number of keypoints, number of feature dimensions).

Parameters:
scene: dict

Scene information as described in Preprocessor documentation

keypointCloud: PCLHelper

Cloud with keypoints
Returns:

tuple with keypoints and features
Returntype:

tuple(np.ndarray[Position_dtype], np.ndarray[FLOAT_dtype, ndim=2])
